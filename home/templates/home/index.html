Technical Implementation (Student Notes)

1. Database Models

I created a Product model with fields for name, description, price, and category. I also added an image_url field to show product pictures dynamically.

2. URL Routing

The project uses nested URL configurations. The main project urls.py includes the home app's urls.py for better organization.

3. Templates

I used template inheritance (base.html) to keep the code DRY (Don't Repeat Yourself) and ensure a consistent layout across all pages.
